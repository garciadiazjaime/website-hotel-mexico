## How to install

- Install [nvm](https://github.com/nvm-sh/nvm/blob/master/README.md) (Node Version Manager).

- Set node versions found on .nvmrc:
```
nvm use
```

- Install packages.
```
npm ci
```

## How to start dev server

```
npm run dev
```
